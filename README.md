# 🌟 Why VirtuoUI?

Most UI libraries only provide pre-built components.

**VirtuoUI** goes beyond that by combining a reusable React component library with an AI-powered component generator, enabling developers to instantly generate, customize, preview, and install production-ready React UI components.

Whether you're building dashboards, landing pages, SaaS products, or admin panels, VirtuoUI helps you ship faster with modern and reusable UI components.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/rksaini1614/VirtuoUI-Library-Project.git
```

---

## Install Dependencies

### Frontend

```bash
cd virtuoui-client
npm install
npm run dev
```

### Backend

```bash
cd virtuoui-server
npm install
npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file inside the **server** directory.

```env
PORT=

MONGODB_URl=

JWT_SECRET_key=

OPENROUTER_API_KEY=

FIREBASE_API_KEY=

RAZORPAY_KEY_ID=

RAZORPAY_KEY_SECRET=

```
